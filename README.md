# 개요

## 기능 소개

해당 프로젝트에는 지뢰찾기 게임을 웹 게임으로 구현하였습니다.  지뢰찾기란 맵에 존재하는 지뢰를 모두 찾아 제거하는 게임입니다. 해당 서비스에 가입한 사용자면 게임을 플레이할 수 있으며, 게임 기록 서버에 저장합됩다.

## 기능 정리

| 이름           | 설명                     |   |
| ------------ | ---------------------- | - |
| 사용자 확인하기     | 사용자가 게임 이용자인지 확인하는 기능  |   |
| 프로필 가져오기     | 사용자의 프로필 정보(닉네임 등) 제공  |   |
| 게임하기         | 지뢰찾기 게임 제공             |   |
| 내 게임이력 조회하기  | 사용자의 승전기록 제공           |   |
